Quality Control Checklist

Pre-Delivery Verification

Common Issues & Solutions

Issue: Can't find comparable sales

Solution: Expand search radius, use older comps (up to 12 months), or use price-per-square-foot from neighboring areas

Issue: Hosting links not working

Solution: Test links in private/incognito browser, check file permissions, use alternative hosting platform

Issue: Audio generation failing

Solution: Simplify script, check ElevenLabs credits, try different voice model, or record manually

Issue: FUB tag updates not saving

Solution: Refresh browser, clear cache, check internet connection, use FUB mobile app as backup
